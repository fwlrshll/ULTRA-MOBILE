## 宝塔面板安装指南

### 1. 连接服务器
1. 使用Putty等SSH工具连接服务器
2. 输入IP地址和端口22
3. 使用root账户登录

### 2. 安装宝塔面板
执行以下命令安装宝塔面板：
bash
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

安装完成后，系统会显示面板访问地址和登录凭证，请妥善保存。

### 3. 配置LNMP环境
1. 访问宝塔面板
2. 选择安装LNMP环境
3. 推荐配置：
   - MySQL 5.7
   - PHP 7.3
4. 建议安装opcache扩展优化性能

## WordPress安装教程

### 1. 准备Web环境
- 通过宝塔面板一键部署LAMP/LNMP环境
- 选择极速安装模式

### 2. 部署WordPress
1. 进入宝塔插件中心
2. 选择"一键部署源码"
3. 找到WordPress点击部署

### 3. 配置网站信息
- 填写域名或服务器IP
- 设置数据库账号密码
- 指定网站根目录

### 4. 完成安装
1. 访问网站进入安装向导
2. 填写数据库连接信息
3. 设置管理员账户
4. 完成安装后登录后台
